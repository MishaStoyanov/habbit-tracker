# habbit-tracker

Трей-віджет для Windows (Electron), який допомагає тримати фокус на звичках, харчуванні та витратах — завжди під рукою, в системному треї.

## Що вміє

- **Шкідливі звички** — лічильник днів без зриву, статистика рецидивів, мотиваційні повідомлення.
- **Корисні звички** — облік тренувань на біговій доріжці (час, швидкість, кроки, калорії).
- **Харчування** — оцінка калорійності страви через ШІ (Gemini) за текстовим описом, або ручне введення БЖУ.
- **Норма калорій** — розрахунок за профілем (стать, вік, зріст, вага, активність, дефіцит) з урахуванням спалених калорій.
- **Гроші** — скільки зекономлено на відмові від шкідливих звичок і скільки витрачено на доставку їжі.

## Стан проєкту

Наразі список звичок **захардкоджений під мене** і не налаштовується через інтерфейс — це особистий інструмент, а не готовий продукт.

## Технології

Electron, ванільний JS/HTML/CSS, локальне зберігання даних у JSON, Gemini API для аналізу їжі.

## Запуск

```
npm install
npm start
```
